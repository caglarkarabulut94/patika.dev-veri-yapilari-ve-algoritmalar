# Proje 2

## Merge Sort Projesi

16,21,11,8,12,22] dizisinin Merge Sort algoritması ile sıralanması şu aşamalarla yapılır:

Dizi ufaltılır:

        [16,21], [11,8], [12,22]

Üstteki ufaltılmış diziler birleştirilir:

        [11,8,16,21], [12,22]

Üstteki diziler tekrar birleştirilir:

        [11,8,12,16,21,22]

Big-O gösterimi:

        O(n*log(n))
